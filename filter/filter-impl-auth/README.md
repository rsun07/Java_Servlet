# filter-auth-impl
This is a practice project that implement auth/log in using servlet filter <br>

How to run it, see [servlet-intro-readme](https://github.com/rsun07/Java_Web/tree/master/servlet-intro#how-to-run-it)

## Expected Behavior
- follow the ui page `localhost:8080/filter_impl_auth_war` to log in using `user name`: `root` and `password`: `root`
- try a wrong user name and password to login
